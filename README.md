# 🌍 Global COVID-19 Analytics PowerBI Dashboard | Power BI

## 📌 Overview
This project analyzes global COVID-19 data to identify trends, patterns, and key insights across countries and continents. It demonstrates end-to-end data analysis from cleaning to visualization.

## 🛠️ Tools & Technologies
- Python (Jupyter Notebook) – Data Cleaning
- MySQL Workbench – Data Analysis
- Power BI – Dashboard Visualization
- DAX – KPI and metric calculations

## 🔄 Workflow

### 1. Data Cleaning (Python)
- Cleaned raw COVID dataset
- Handled missing values
- Standardized column formats

### 2. SQL Analysis (MySQL)
- Identified top affected countries
- Analyzed death and recovery trends
- Performed continent-wise comparisons

### 3. Dashboard (Power BI)
- Created KPI cards (Total Cases, Deaths, Recovery Rate)
- Built insight cards (Top Country, Death Rate, Recovery Rate)
- Designed interactive visuals

## 📊 Key Insights
- Total Cases: 19M
- Total Deaths: 792K
- Recovery Rate: 63%
- Most Affected Country: USA
- Highest Death Rate: Vatican City
- Highest Recovery Rate: Dominica
